# Reports Section

## Overview
The Reports section provides comprehensive analytics for business insightsâ€”revenue reports with session/inventory breakdown, resource utilization analysis, customer insights, and a complete operation history log with advanced filtering.

## Components

- **ReportsPage** - Main reports dashboard with all report sections
- **RevenueReport** - Revenue summary and charts
- **UtilizationReport** - Resource utilization statistics
- **OperationHistory** - Chronological operation log

## User Flows

- View daily revenue breakdown (sessions vs inventory)
- View weekly and monthly revenue summaries
- Compare revenue between periods
- View top customers by revenue
- View resource utilization rates
- Identify peak usage hours
- View average session duration
- View customer acquisition trends
- Browse operation history log
- Filter operations by type, date, customer, or resource
- Export reports to CSV/PDF

## UI Requirements

- Revenue dashboard with total revenue card, breakdown chart, trend line chart, top customers list
- Utilization dashboard with overall percentage, resource bar chart, peak hours heatmap
- Customer insights with new customers trend, type distribution, retention rate
- Operation history with chronological log table, type filter, date range picker
- Export buttons for each report section
- Date range selector for all reports
