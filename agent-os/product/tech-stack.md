# Tech Stack

## Frontend

- **React 19** — مع TypeScript
- **Tailwind CSS v4** — للتصميم
- **Vite** — للـ Build tooling
- **Zustand** — للـ State Management
- **React Hook Form + Zod** — للـ Form validation
- **Radix UI** — للـ Accessible components
- **Lucide React** — للـ Icons
- **React Hot Toast** — للـ Notifications

## Backend (Desktop)

- **Tauri v2** — للـ Desktop App
- **Rust** — للـ Backend logic
- **SQLite** — للـ Local database (قادم)

## Design System

- **Colors**: Stone palette (warm neutrals)
- **Typography**: Noto Sans (Arabic + Latin)
- **Direction**: RTL-first with CSS Logical Properties

## Development Tools

- **Bun** — Package manager
- **ESLint** — Linting
- **Lefthook** — Git hooks
- **TypeScript** — Type safety
